CODE FICANDO

Do físico para o digital!







Mestre em Ciência da Computação pela UFPB, Andrés Menéndez, sentiu a necessidade de ampliar a performance de sala de aula para o ambiente virtual. Considerando vídeos no YouTube como material de suporte tanto para seus alunos da UFS, onde é professor. Podendo também ajudar outras pessoas com seus conhecimentos.

O desenvolvimento do briefing foi focado no horizonte das tecnologias digitais, considerando a receptividade dos alunos e dos seguidores no canal, se referenciando em um ambiente de ensino saudável e ágil com amplitude global.




Como o projeto de uma grande dimensão, pois é divulgado no YouTube, foi optado pelo desenvolvimento de matérias gráficos impressos e digitais para a marca. Assim, foi feito um material de branding para o canal, não ficando apenas o material digital.

O branding desenvolvido para a marca foi composto de material impresso: cartão de visita, timbrado, ofício e envelopes, assim como material impresso para estampagem de copos. Foi considerado para a construção desse material, uma disseminação efetiva seja ela física ou digital, uma vez que o canal seria também disseminado na Universidade Federal de Sergipe.










Material do Branding da Marca e Vinheta

Para o desenvolvimento da vinheta, que tinha requisito de ser breve, considerei a naturalidade de “codar” considerando a ênfase da animação para o termo code, uma vez que escrevemos diversas linhas de código. E o destaque do CODE reafirmando o foco principal do canal.

Quando se trata do YouTube, fiz todo o material para está plataforma, como: logo inicial, banner, as thumbnails para os vídeos. Como os vídeos são desenvolvidos com base em apresentação de slides, também foram feitos templates para cada linguagem abordada nos vídeos.
